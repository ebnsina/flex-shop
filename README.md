# Flex Shop


### Features
- Register
- Confirm Register
- Email Login
- Google Login
- Reset Password
- Category
- Subcategory
- Add To Cart
- View Detail, Description
- Out of Stock
- Add To Wishlist
- Add Rating
- Filter by Category, Subcategory, Brand, Color, Shipping, Price
- Payment: Stripe, COD 
- Coupon Discount


## Frontend
- React
- Redux


## Backend
- Node.js (Express.js)
